# ratatui-recipe template

This directory contains a cargo-generate template for creating new projects based on the ratatui-recipe example app.

Usage

Prerequisites:

- Install cargo-generate: `cargo install cargo-generate`

Generate a new project from this template:

```sh
cargo generate --git https://github.com/ls3205/ratatui-recipe.git --path template --name my-app
```

For more details about the base project see the repository: https://github.com/ls3205/ratatui-recipe.git
